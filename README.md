Project Description:

Slot machine is a children's game with Paw Patrol theme. It has 3 reels with 3 paw patrol characters per reel. Gamers have an initial 500 coins to start playing with. They can bet 5 or 50 coins each turn. If the gamer runs out of coins an 'alert' shows to insert more coins. If the gamer hits a jackpot (all reels shows the same paw patrol character), the gamer wins more coins to continue playing the game.

Project made with CSS, HTML and Javascript.
